KeyBank Project

The KeyBank project is a secure, access-control prototype designed to manage and monitor physical keys in workplace environments. The system integrates fingerprint authentication, RFID verification, and door-specific PIN codes to ensure that only authorized personnel can retrieve designated keys. Upon successful validation, KeyBank grants access to the appropriate key container, records the transaction, and dispatches automated SMS/e-mail notifications to the person and registered administrators for real-time oversight. The intent behind this project is to improve security and accountability within automotive dealerships while streamlining daily operations through a modular, efficient, and easily expandable workflow. This report outlines the project's hardware architecture, software implementation, operational workflow, and performance results, demonstrating its potential as a reliable and scalable key-management solution.
