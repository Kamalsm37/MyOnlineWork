Project Description

The KeyBank project is a secure access-control system designed to manage and monitor physical keys in workplace environments. It integrates fingerprint authentication, RFID verification, and PIN-based security to ensure only authorized personnel can access specific key containers.

Once authentication is successful, the system unlocks the corresponding key slot, logs the transaction, and sends automated notifications to administrators via SMS and email. This ensures full accountability and real-time monitoring of key usage.

The project combines embedded hardware with a web-based interface to create a scalable and efficient key management solution, primarily designed for automotive dealerships and secure facilities.
